# Feature Scope

## Core features

- daily journal writing
- built-in calendar
- markdown editing
- tags and page links
- page views with derived sections
- hierarchical page navigation
- search
- timeline and calendar navigation
- tasks
- assets and attachments
- local derived index/cache
- cross-device sync

## Strong optional features

- graph visualization
- PDF workflows
- whiteboards
- flashcards
- plugin APIs
- compatibility import / convenience export
- templates
- quick capture

## Features to avoid initially

- block refs and embeds
- advanced block query language
- collaborative editing
- complex database abstraction exposed to users
- plugin API that can mutate storage directly
