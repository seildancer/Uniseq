# React and Vite

React is the UI layer.

Vite is the development and build tool for the frontend bundle.

## Responsibilities

- route rendering
- editor integration
- command palette
- sidebar and panels
- settings
- graph and visual feature components
- plugin host UI
- feature orchestration

## State model

Use frontend state for:

- active route
- selected date/page
- editor focus
- panel layout
- modal state
- temporary form state
- cached query responses

Do not use frontend state as the source of truth for workspace content.
