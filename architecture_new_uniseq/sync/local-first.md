# Local-First Storage

Local markdown files are canonical.

The app should work without login and without network.

## Local guarantees

- open workspace
- write journal
- browse pages
- search local index
- use tasks and views
- export data

Sync adds cross-device continuity, not basic functionality.

Cache rebuilds must also be local-first.

Deleting or rebuilding local caches must never require network access and must never risk user markdown content.
