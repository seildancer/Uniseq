# Journal

The journal is the default screen.

## Required behavior

- open today's journal instantly
- previous/next date navigation
- calendar jump
- built-in month/week calendar access
- quick append
- tag and page autocomplete
- task checkbox support
- attach asset
- show related pages from today's tags

## Design rule

The journal should feel like the place where work happens, not just an inbox to clean later.

Calendar should feel like part of the journal system, not like a detachable feature.
