# Rendering and Media

The app should treat rendering as a first-class capability of the reading and editing experience.

## Core rendering areas

- markdown formatting
- code blocks
- math rendering
- images and media embeds
- lightbox behavior for images
- asset previews where appropriate

## Design rule

Rendering enhancements should not require a graph-native data model.

They are presentation capabilities layered on top of markdown content and asset references.
